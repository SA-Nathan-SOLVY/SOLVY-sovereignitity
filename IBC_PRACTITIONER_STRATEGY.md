## 💡 THE VALUE PROPOSITION FOR IBC PRACTITIONERS

### Current Pain Point
IBC practitioners with whole life policies face friction:
1. **Request policy loan** from carrier (paperwork, delays)
2. **Receive check or ACH** (3-5 business days)
3. **Deposit to external bank** (another institution)
4. **Track loan separately** from banking
5. **Pay interest** to own policy (good) but friction (bad)

### SOLVY Solution: One-Click Bridge Loans
**Vision:** As easy as OneAmerica's unenrolled deposits to PUA

```
IBC Practitioner Flow:
┌─────────────────────────────────────────────────────────────┐
│  1. Log into SOLVY Ecosystem™                               │
│  2. View policy cash value (linked via API)                │
│  3. Enter loan amount → Click "Bridge to SOLVY Card™"      │
│  4. Tax-free loan processed (same day)                     │
│  5. Funds available instantly on SOLVY Card™               │
│  6. Use for business/personal (tap-to-pay anywhere)        │
│  7. Track loan + spending in one dashboard                 │
└─────────────────────────────────────────────────────────────┘
```

**Key Benefits:**
- ⚡ **Instant availability** (vs 3-5 days)
- 💳 **Spend immediately** (vs deposit then transfer)
- 📊 **Unified tracking** (policy + card in one view)
- 🏦 **Member-owned bank** (vs Wells Fargo/Chase)
- 💰 **Profits flow back** (70% to Member Pool)

## 🏗️ THE BRIDGE LOAN PROCESS

### Phase 1: Manual Bridge (Months 1-6)

**Process:**
1. IBC practitioner applies for SOLVY Card™
2. Completes standard onboarding (KYC)
3. Requests "IBC Bridge" feature during setup
4. Provides policy details (carrier, policy #, cash value)
5. **Manual verification** (calls carrier, confirms CV)
6. Practitioner requests policy loan from carrier
7. Loan proceeds ACH to SOLVY account
8. **Same-day availability** on SOLVY Card™

**Timeline:** 1-2 business days (vs 5-7 currently)

**Carriers Supported:**
- OneAmerica (priority - your existing relationship)
- MassMutual
- Guardian Life
- New York Life
- Mutual Trust Life
- Lafayette Life

### Phase 2: Semi-Automated (Months 6-12)

**Process:**
1. IBC practitioner links policy via secure API
2. Real-time cash value displayed in SOLVY app
3. Requests loan amount (up to 95% of CV)
4. SOLVY system **pre-fills loan request** to carrier
5. Practitioner reviews & e-signs
6. Carrier processes loan (API integration)
7. Funds automatically deposited to SOLVY
8. **Instant availability** on SOLVY Card™

**Timeline:** Same day (morning request = afternoon availability)

### Phase 3: One-Click (Year 2+)

**Process:**
1. IBC practitioner views available credit
2. Enters amount → Clicks "Bridge Now"
3. Biometric confirmation (Face ID/Touch ID)
4. SOLVY system processes loan via carrier API
5. Funds instantly available on SOLVY Card™
6. **As easy as OneAmerica PUA deposits**

**Timeline:** Instant (< 30 seconds)
