Minimal CSS to Group Navbar (Without Changing Content)
If your existing page has a navbar or header section, you can add this CSS to conceptually group the links/buttons without moving or redesigning anything. It just adds visual separation.

css
/* Conceptual grouping – no content changes */
.nav-group-left, .nav-group-center, .nav-group-right {
    display: inline-flex;
    gap: 1.5rem;
    align-items: center;
}

.nav-group-left { /* no extra style */ }
.nav-group-center { border-left: 1px solid #eef2f6; border-right: 1px solid #eef2f6; padding: 0 1.5rem; }
.nav-group-right { }
Apply to your existing navbar links by wrapping them in <div class="nav-group-left">, <div class="nav-group-center">, <div class="nav-group-right">. Your existing content stays exactly the same.
